# Face Detection

### To run the app follow the steps:

```
python3 -m venv venv 
source venv/bin/activate
pip3 install -r requirements.txt
python3 face_detection_webcam.py
```

or

```
python3 -m venv venv 
source venv/bin/activate
pip3 install -r requirements.txt
python3 face_detection_image.py
```
